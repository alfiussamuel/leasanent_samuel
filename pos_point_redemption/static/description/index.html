POS Loyalty Management Module

Loyalty management is widely used in the retail industries. A loyalty card, rewards card, points card, advantage card, or club card is a plastic or paper card, visually similar to a credit card or debit card, that identifies the card holder as a member in a loyalty program. Such cards are used to provide redeemable points earned against the purchases. Earned points can be redeemed against the future purchase for equivalent amount.

Points Scheme Release

Points Scheme Release allows user to defined different points earning scheme between different date span. Points scheme can be made applicable in two ways as follows

A) On Sales Invoice

B) Points/ Product/ Product Category

Redemption Points

This will allow user to define points redemption rule. User can define no. of points equivalent to one unit of currency. System will compute amount to be discounted from the bill amount based on no. of points redeemed.

Payment Method

User can maintain different payment modes like cash, bank, etc. Here we have created "Point Redemption" payment method, so lets check it. There is one boolean fields added on this panel called "For Points". When click on such payment methods, system will generate additional field on payment screen to capture no. of points to be redeemed, this will compute discount amount to be reduced from total bill amount against points.

In user form need to define Point Redemption payment method in Available Payment Methods.

POS Transaction

User can sale products using POS screen. Each user will have dedicated POS terminal. Based on the configuration made for points scheme release, system will show message at the bottom of payment screen and same is printed on payment receipt and the points will be credited against selected customer name. If customer name is not provided then such points shall not be taken in to account.

Customer wise point tracking

System maintains track of customer wise transaction in order to calculate closing balance of redeemable points. User can view transaction wise logged in Customer Partner.
