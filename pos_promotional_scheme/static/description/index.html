POS Promotional Schemes

Product promotional schemes widely used in retail industry and has great impact on total sales turnover of business. This module helps user to create different promotional schemes that suits to their business. Promotional schemes created in Odoo shall be utilize in POS module so that billing will be generated based on it.

Promotional schemes can be configured for

  • Specific product category/products
  • Specific date range
  • Different locations/branches

Promotional scheme allows user to define different product promotional schemes for different date span.

Buy X and get Y free

User can define scheme name here and select scheme basis is "On Same Product". This scheme will be applicable on selected product only.

Buy X and get Y free

User can define scheme name and need to select scheme basis is "On Different Product". This scheme will be available on the selected products. User has to define scheme available on product and scheme product.

On Unit Price Discount (Amount)

User can define scheme type is "Unit Price Discount (Amount)". Then user has to define the slab for this scheme and offer is quantity wise unit price discount for selected products.

On Unit Price Discount (Percentage)

User can define scheme type is "Unit Price Discount (Percentage)". Then user has to define the slab for this scheme and offer is quantity wise discount in % on sales price of selected products.

On Volume Discount

User can define scheme type is "Unit Price Discount (Percentage)". Here user has to define buying quantity and offer price.

Promotional Scheme POS Transactions

Once the mentioned promotional schemes are created in system then they can be used in POS transaction. Here customer will choose products and bring them to a billing counter. User will scan or add each of the product on billing counter and then will click on "Check Scheme" option. This action will rearrange all the products based on the scheme available and will apply discounts on it.
